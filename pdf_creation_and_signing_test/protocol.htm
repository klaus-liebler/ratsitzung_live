Dies ist ein Testtext für die Markdown-zu-PDF-Umwandlung.

Der folgende Inhalt ist absichtlich laenger gehalten, damit der PDF-Generator sauber über mehrere Seiten umbricht und der Footer unter Realbedingungen getestet werden kann.

Drucklayout-Hinweis

Die CSS-Werte im HTML-Template nutzen bereits Druckeinheiten in Millimeter. Beispiele:

  • @page { margin: 12mm 15mm 18mm 15mm; }

  • body { margin: 0; }

  • h1 { margin-top: 0; margin-bottom: 6mm; }

  • table { margin-top: 4mm; } Wenn du mehr Abstand zum Seitenende brauchst, sind diese Varianten typisch:

  • body { margin: 0 15mm 22mm 15mm; }

  • @page { margin: 14mm 15mm 20mm 15mm; }

  • h2 { break-after: avoid; }

  • table { page-break-inside: auto; }

Weitere Hinweise für den Druck

Wenn du mehr Abstand zum Seitenende brauchst, sind diese Varianten typisch:

  • body { margin: 0 15mm 22mm 15mm; }
  • @page { margin: 14mm 15mm 20mm 15mm; }
  • h2 { break-after: avoid; }
  • table { page-break-inside: auto; }

Für ganz enge Layouts kann man die untere Seitenmarge kleiner machen, muss dann aber den Footer inhaltlich weiter nach unten oder die Trennlinie weiter nach unten setzen.
